# MIMIR - DOCX JSON Converter pour VSCode/Cursor

Extension pour convertir des documents DOCX en JSON et HTML, spécialement conçue pour traiter des composants pédagogiques.

## Fonctionnalités

- Conversion de documents DOCX en HTML avec support pour les composants pédagogiques
- Conversion de documents DOCX en JSON pour le traitement ultérieur
- Prévisualisation des documents DOCX convertis directement dans l'éditeur

## Configuration

- `docxJsonConverter.pythonPath`: Chemin vers l'exécutable Python
- `docxJsonConverter.scriptPath`: Chemin vers le script de conversion DOCX
- `docxJsonConverter.saveImagesToDisk`: Si vrai, sauvegarde les images sur disque plutôt que de les encoder en base64

## Utilisation

1. Clic droit sur un fichier DOCX dans l'explorateur
2. Sélectionnez "DOCX: Convertir en HTML", "DOCX: Convertir en JSON" ou "DOCX: Prévisualiser en HTML"
